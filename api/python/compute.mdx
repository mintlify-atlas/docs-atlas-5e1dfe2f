## Overview

The compute module provides functions for performing computations on Arrow data structures. These functions support both scalar operations and aggregations.

```python
import pyarrow as pa
import pyarrow.compute as pc

arr = pa.array([1, 2, 3, 4, 5])
result = pc.sum(arr)
print(result)  # 15
```

## Arithmetic Functions

### add()

Add two arrays or scalars element-wise.

```python
pc.add(x, y, memory_pool=None)
```

<ParamField path="x" type="Array-like or scalar-like">
  First argument.
</ParamField>

<ParamField path="y" type="Array-like or scalar-like">
  Second argument.
</ParamField>

<ParamField path="memory_pool" type="pyarrow.MemoryPool" default="None">
  Memory pool for allocation.
</ParamField>

<ResponseField name="result" type="Array or Scalar">
  Element-wise sum of x and y.
</ResponseField>

### subtract()

Subtract two arrays or scalars element-wise.

```python
pc.subtract(x, y, memory_pool=None)
```

### multiply()

Multiply two arrays or scalars element-wise.

```python
pc.multiply(x, y, memory_pool=None)
```

### divide()

Divide two arrays or scalars element-wise.

```python
pc.divide(x, y, memory_pool=None)
```

### power()

Raise to power element-wise.

```python
pc.power(base, exponent, memory_pool=None)
```

<ParamField path="base" type="Array-like or scalar-like">
  Base value.
</ParamField>

<ParamField path="exponent" type="Array-like or scalar-like">
  Exponent value.
</ParamField>

<ResponseField name="result" type="Array or Scalar">
  Element-wise power computation.
</ResponseField>

### negate()

Negate values element-wise.

```python
arr = pa.array([1, -2, 3])
result = pc.negate(arr)
print(result)  # [-1, 2, -3]
```

### abs()

Compute absolute value element-wise.

```python
arr = pa.array([-1, -2, 3])
result = pc.abs(arr)
print(result)  # [1, 2, 3]
```

## Aggregation Functions

### sum()

Compute sum of array values.

```python

```

<ParamField path="array" type="Array-like">
  Array to aggregate.
</ParamField>

<ParamField path="skip_nulls" type="bool" default="True">
  Whether to skip null values.
</ParamField>

<ParamField path="min_count" type="int" default="1">
  Minimum number of non-null values required.
</ParamField>

<ResponseField name="result" type="Scalar">
  Sum of array values.
</ResponseField>

### mean()

Compute arithmetic mean.

```python
arr = pa.array([1, 2, 3, 4, 5])
result = pc.mean(arr)
print(result)  # 3.0
```

### min()

Find minimum value.

```python
pc.min(array, skip_nulls=True, min_count=1, memory_pool=None)
```

### max()

Find maximum value.

```python
pc.max(array, skip_nulls=True, min_count=1, memory_pool=None)
```

### count()

Count non-null values.

```python

```

<ParamField path="array" type="Array-like">
  Array to count.
</ParamField>

<ParamField path="mode" type="str" default="'only_valid'">

<ResponseField name="result" type="Scalar">

### variance()

Compute variance.

```python
pc.variance(array, ddof=0, skip_nulls=True, min_count=0, memory_pool=None)
```

### stddev()

## Filter and Selection

### filter()

Filter values based on a boolean mask.

```python
pc.filter(data, mask, null_selection_behavior='drop', memory_pool=None)
```

<ParamField path="data" type="Array, ChunkedArray, RecordBatch, or Table">
  Data to filter.
</ParamField>

<ParamField path="mask" type="Array or Expression">
  Boolean mask for filtering.
</ParamField>

<ParamField path="null_selection_behavior" type="str" default="'drop'">
  How to handle nulls in mask: 'drop' or 'emit_null'.
</ParamField>

<ResponseField name="result" type="Same as input type">
  Filtered data.
</ResponseField>

### take()

Select values by indices.

```python
pc.take(data, indices, boundscheck=True, memory_pool=None)
```

<ParamField path="data" type="Array, ChunkedArray, RecordBatch, or Table">
  Data to select from.
</ParamField>

<ParamField path="indices" type="Array, ChunkedArray">
  Integer indices to select. Must be integer type.
</ParamField>

<ParamField path="boundscheck" type="bool" default="True">
  Whether to check indices are in bounds.
</ParamField>

<ResponseField name="result" type="Same as input type">
  Selected values.
</ResponseField>

### index()

Find index of first occurrence of a value.

```python
pc.index(data, value, start=None, end=None, memory_pool=None)
```

<ParamField path="data" type="Array-like">
  Array to search in.
</ParamField>

<ParamField path="value" type="Scalar-like">
  Value to search for.
</ParamField>

<ParamField path="start" type="int" default="None">
  Start index for search.
</ParamField>

<ParamField path="end" type="int" default="None">
  End index for search.
</ParamField>

<ResponseField name="index" type="int">
  Index of first occurrence, or -1 if not found.
</ResponseField>

<ResponseField name="indices" type="UInt64Array">

</ResponseField>

### bottom_k_unstable()


## Expression API

The Expression API provides a way to build complex compute expressions.

### field()

Reference a field in a dataset.

```python
pc.field('column_name')
pc.field(('nested', 'field'))
pc.field(0)  # By index
```

<ParamField path="name_or_index" type="str, tuple, or int">
  Field name, nested field tuple, or column index.
</ParamField>

<ResponseField name="expr" type="Expression">
  Field reference expression.
</ResponseField>

### scalar()

Create a scalar expression.

```python
expr = pc.scalar(42)
expr = pc.scalar('hello')
```

<ParamField path="value" type="bool, int, float, or str">
  Python value to convert to expression.
</ParamField>

<ResponseField name="expr" type="Expression">
  Scalar value expression.
</ResponseField>

### Example: Complex Expressions

```python
import pyarrow as pa
import pyarrow.compute as pc
import pyarrow.dataset as ds

# Create a filter expression
filter_expr = (pc.field('age') > 18) & (pc.field('income') < 100000)

# Use with dataset filtering
table = pa.table({
    'age': [25, 15, 35, 45],
    'income': [50000, 0, 80000, 120000]
})

filtered = table.filter(filter_expr)
print(filtered)
```

## Function Registry

```python

```

<ResponseField name="functions" type="list of str">
  Names of all available functions.

### get_function()

```python

```

<ParamField path="name" type="str">
  Function name.
</ParamField>

<ResponseField name="function" type="Function">
  The compute function object.
</ResponseField>

### call_function()

<ParamField path="name" type="str">
  Function name to call.

<ParamField path="args" type="list">
  Function arguments.
</ParamField>

<ParamField path="options" type="FunctionOptions" default="None">
  Function-specific options.
</ParamField>

<ResponseField name="result" type="Array or Scalar">
  Function result.
</ResponseField>
