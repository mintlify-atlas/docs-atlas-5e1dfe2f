## Table

A Table is a two-dimensional dataset with chunked arrays for columns.

```python
import pyarrow as pa

data = {
    'n_legs': [2, 4, 5, 100],
    'animals': ['Flamingo', 'Horse', 'Brittle stars', 'Centipede']
}
table = pa.table(data)
print(table)
```

### table()

Create a pyarrow.Table from Python data.

```python
pa.table(data, schema=None, metadata=None)
```

<ParamField path="data" type="dict, list, pandas.DataFrame">
  Dictionary of column names to arrays/lists, list of arrays, or pandas DataFrame.
</ParamField>

<ParamField path="schema" type="pyarrow.Schema" default="None">
  If provided, dictates the schema of the table. Otherwise inferred.
</ParamField>

<ParamField path="metadata" type="dict" default="None">
  Optional metadata for the schema.
</ParamField>

<ResponseField name="table" type="pyarrow.Table">
  A Table constructed from the input data.
</ResponseField>

### Properties

#### schema

Return the table's schema.

```python
table = pa.table({'a': [1, 2], 'b': [3, 4]})
print(table.schema)
# a: int64
# b: int64
```

<ResponseField name="schema" type="pyarrow.Schema">
  The table's schema.
</ResponseField>

#### num_rows

Number of rows in the table.

```python
table = pa.table({'a': [1, 2, 3]})
print(table.num_rows)  # 3
```

<ResponseField name="num_rows" type="int">
  The number of rows.
</ResponseField>

#### num_columns

Number of columns in the table.

```python
table = pa.table({'a': [1, 2], 'b': [3, 4]})
print(table.num_columns)  # 2
```

<ResponseField name="num_columns" type="int">
  The number of columns.
</ResponseField>

#### column_names

List of column names.

```python
table = pa.table({'a': [1, 2], 'b': [3, 4]})
print(table.column_names)  # ['a', 'b']
```

<ResponseField name="column_names" type="list of str">
  The column names as a list.
</ResponseField>

#### columns

List of all columns as ChunkedArray objects.

```python
table = pa.table({'a': [1, 2], 'b': [3, 4]})
for col in table.columns:
    print(col)
```

<ResponseField name="columns" type="list of ChunkedArray">
  The columns as ChunkedArray objects.
</ResponseField>

<ResponseField name="table" type="pyarrow.Table">

<ResponseField name="table" type="pyarrow.Table">

#### to_pydict()

Convert table to a Python dictionary.

```python

```

<ResponseField name="dict" type="dict">
  A dictionary with column names as keys and Python lists as values.
</ResponseField>

#### to_pandas()

```python

```

<ParamField path="self_destruct" type="bool" default="False">

</ParamField>

<ResponseField name="dataframe" type="pandas.DataFrame">
  A pandas DataFrame with the table data.
</ResponseField>

<ResponseField name="table" type="pyarrow.Table">

## RecordBatch

A RecordBatch is a collection of equal-length arrays.

```python
import pyarrow as pa

data = [
    pa.array([1, 2, 3]),
    pa.array(['a', 'b', 'c'])
]
schema = pa.schema([('nums', pa.int64()), ('letters', pa.string())])
batch = pa.record_batch(data, schema=schema)
print(batch)
```

### record_batch()

Create a RecordBatch from arrays or a dictionary.

```python
pa.record_batch(data, schema=None, metadata=None)
```

<ParamField path="data" type="list of Array, dict">
  List of arrays or dictionary of column names to arrays.
</ParamField>

<ParamField path="schema" type="pyarrow.Schema" default="None">
  Schema defining the batch structure.
</ParamField>

<ParamField path="metadata" type="dict" default="None">
  Optional metadata.
</ParamField>

<ResponseField name="batch" type="pyarrow.RecordBatch">
  A RecordBatch with the specified data.
</ResponseField>

## Utility Functions


## RecordBatchReader

Streaming reader for record batches.

```python
import pyarrow as pa

# Create a reader from batches
batches = [pa.record_batch([[1, 2], [3, 4]], names=['a', 'b'])]
reader = pa.RecordBatchReader.from_batches(batches[0].schema, batches)

for batch in reader:
    print(batch)
```

### from_batches()

Create a reader from a list of record batches.

```python
RecordBatchReader.from_batches(schema, batches)
```

<ParamField path="schema" type="pyarrow.Schema">
  Schema of the batches.
</ParamField>

<ParamField path="batches" type="iterable of RecordBatch">
  Batches to read.
</ParamField>

<ResponseField name="reader" type="pyarrow.RecordBatchReader">
  A batch reader.
</ResponseField>

### read_all()

Read all batches and return as a Table.

```python
reader.read_all()
```

<ResponseField name="table" type="pyarrow.Table">
  All batches combined into a table.
</ResponseField>

### read_next_batch()

Read the next batch.

```python
reader.read_next_batch()
```

<ResponseField name="batch" type="pyarrow.RecordBatch">
  The next batch, or raises StopIteration if exhausted.
</ResponseField>
